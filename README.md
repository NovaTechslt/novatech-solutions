# NovaTech Solutions

Site estático pronto para Cloudflare Pages.

## Publicar no GitHub
1. Abra o repositório `novatech-solutions`.
2. Clique em **Add file > Upload files**.
3. Arraste todos os arquivos desta pasta.
4. Clique em **Commit changes**.

## Cloudflare Pages
- Framework preset: **None**
- Build command: deixe vazio
- Build output directory: `/`

## Catálogo
Edite `products.json`. O primeiro item contém o link teste `https://meli.la/1a57Ccd`; confirme o produto antes da publicação.
